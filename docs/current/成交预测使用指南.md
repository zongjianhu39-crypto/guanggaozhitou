# 成交预测功能使用指南

## 功能概述

成交预测功能基于 LightGBM 机器学习模型，预测不同人群在不同日期的：
- **成交概率**：该人群当天是否会有成交（分类模型，AUC=0.917）
- **订单成本**：有成交时的订单成本（回归模型，R²=0.154）
- **最终预测成本** = 成交概率 × 订单成本

## 访问入口

### 方式1：导航栏
点击顶部导航栏的 **"成交预测"** 链接

### 方式2：首页入口
在首页"系统入口"区域，点击 **"成交预测"** 卡片

## 使用步骤

### 1. 选择日期范围
- 默认显示本月日期范围
- 点击日期选择器修改开始和结束日期

### 2. 筛选条件（可选）
- **人群名称**：输入关键词筛选特定人群
- **场景**：从下拉列表选择特定场景

### 3. 运行预测
点击 **"🔮 运行预测"** 按钮，等待模型计算

### 4. 查看结果

#### 统计概览
- 预测人群数
- 平均成交概率
- 高概率人群数量（>60%）
- 预测总成本

#### 详细表格
| 列名 | 说明 |
|------|------|
| 日期 | 预测日期 |
| 场景 | 广告场景名称 |
| 人群名称 | 目标人群名称 |
| 成交概率 | 绿色(>60%) / 橙色(30-60%) / 红色(<30%) |
| 订单成本 | 预测的订单成本（花费/成交笔数） |
| 预测总成本 | 成交概率 × 订单成本 |
| 置信区间 | 预测的不确定性范围 |

### 5. 排序和分页
- 使用排序下拉菜单按不同字段排序
- 使用分页按钮浏览更多结果（每页50条）

### 6. 导出结果
点击 **"📥 导出CSV"** 按钮下载预测结果

## 使用建议

### 人群筛选
- **优先投放**：成交概率 > 60% 的人群
- **谨慎投放**：成交概率 30-60% 的人群
- **谨慎评估**：成交概率 < 30% 的人群

### 预算分配
- 高成交概率人群分配更多预算
- 关注预测成本过高的人群，评估ROI

### 效果追踪
- 定期回填实际成本到 `model_predictions` 表
- 评估模型预测准确性
- 根据实际效果调整投放策略

## 注意事项

1. **模型局限性**
   - 基于历史数据训练，新人群可能预测不准
   - 置信区间较宽表示预测不确定性高
   - 建议结合实际业务经验综合判断

2. **数据要求**
   - 需要足够的历史数据（至少7天）
   - 新场景或新人群缺少历史特征时预测效果差

3. **性能优化**
   - 大范围日期预测可能需要较长时间
   - 建议使用筛选条件缩小范围

## 故障排查

### 预测失败：没有数据
- 检查日期范围是否正确
- 确认该日期范围内有投放数据
- 检查筛选条件是否过于严格

### 预测结果异常
- 查看置信区间，区间宽表示不确定性高
- 检查人群是否为新增人群（缺少历史数据）
- 对比历史实际成本

### 导出CSV乱码
- CSV文件使用UTF-8编码（带BOM）
- 使用Excel或Numbers打开
- 如仍有问题，用文本编辑器打开并另存为UTF-8
