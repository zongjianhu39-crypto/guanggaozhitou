# 成交预测功能使用指南

## 功能概述

成交预测功能已升级为货盘人群推荐：导入当天达人直播间货盘后，系统会结合商品结构、历史人群表现、近期活跃度和成本模型，推荐更适合测试和放量的人群。

## 访问入口

### 方式1：导航栏
点击顶部导航栏的 **"成交预测"** 链接

### 方式2：首页入口
在首页"系统入口"区域，点击 **"成交预测"** 卡片

## 使用步骤

### 1. 选择推荐日期
- 默认显示今天
- 可按实际直播日期修改

### 2. 导入当天货盘
- 上传 CSV 文件，或直接粘贴 CSV 内容
- 支持字段：商品id、商品名称、分类/类目/品类、单价/价格、佣金率、支付GMV、支付订单数、支付销量、退货率等

### 3. 生成推荐
点击 **"生成推荐"** 按钮，等待系统计算 Top10/Top20/Top30 推荐人群

### 4. 查看结果

#### 统计概览
- 推荐人群数
- 平均匹配分
- 优先测试以上人群数
- 平均预估ROI

#### 详细表格
| 列名 | 说明 |
|------|------|
| 排名 | 推荐排序 |
| 人群名称 | 目标人群名称 |
| 推荐等级 | 强推 / 优先测试 / 小预算测试 / 观察 |
| 匹配分 | 综合货盘、历史表现、近期活跃度后的排序分 |
| 预估转化 | 预测转化参考值 |
| 预估ROI | 预测ROI参考值 |
| 置信度 | 历史样本和近期活跃度带来的可信程度 |
| 推荐理由 | 进入推荐的主要原因 |

### 5. 排序和分页
- 使用排序下拉菜单按匹配分、ROI、置信度等字段排序
- 使用分页按钮浏览更多结果（每页50条）

### 6. 导出结果
点击 **"导出CSV"** 按钮下载推荐结果

## 使用建议

### 人群筛选
- **优先测试**：匹配分高且置信度高的人群
- **小预算测试**：货盘相关但样本少的人群
- **观察**：匹配分或近期活跃度较低的人群

### 预算分配
- 高匹配、高置信度人群可正常预算测试
- 低样本但相关的人群建议小预算探测

### 效果追踪
- 每天记录推荐排名、是否实际投放、花费、成交、ROI、CPO
- 7天后对比人工选择和模型推荐效果
- 根据实际效果调整投放策略

## 注意事项

1. **模型局限性**
   - 基于历史数据训练，新人群可能预测不准
   - 置信区间较宽表示预测不确定性高
   - 建议结合实际业务经验综合判断

2. **数据要求**
   - 需要足够的历史数据（至少7天）
   - 新场景或新人群缺少历史特征时预测效果差

3. **性能优化**
   - 大范围日期预测可能需要较长时间
   - 建议使用筛选条件缩小范围

## 故障排查

### 预测失败：没有数据
- 检查日期范围是否正确
- 确认该日期范围内有投放数据
- 检查筛选条件是否过于严格

### 预测结果异常
- 查看置信区间，区间宽表示不确定性高
- 检查人群是否为新增人群（缺少历史数据）
- 对比历史实际成本

### 导出CSV乱码
- CSV文件使用UTF-8编码（带BOM）
- 使用Excel或Numbers打开
- 如仍有问题，用文本编辑器打开并另存为UTF-8
